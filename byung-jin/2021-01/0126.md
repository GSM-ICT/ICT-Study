낡은 종속성 버그 등 다양한 시행착오가 있었습니다.

```shell
cd D:\TIL-Book

gitbook install && gitbook build

cp -R _book/* .

git clean -fx node_modules
git clean -fx _book

git add .

git commit -a -m "Update github pages"

git push origin main
```
