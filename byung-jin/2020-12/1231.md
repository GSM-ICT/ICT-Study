## 프리즈마

Prisma를 사용해야하는 입장이 되어, 예제를 보며 천천히 따라했다.
모델을 만드는 규칙정도를 알게되었다.

```prisma
datasource db {
  provider = "mysql"
  url      = env("DATABASE_URL")
}

generator client {
  provider = "prisma-client-js"
}

model User{
  id  Int  @default(autoincrement()) @id
  email String? @unique
  name String
  age Int
}

model Book{
  id  Int @default(autoincrement()) @id
  title String
  description String
}

model Loans{
  id Int @default(autoincrement()) @id
  user User @relation(fields: [userId], references: [id])
  userId Int @unique
  book Book @relation(fields: [bookId], references: [id])
  bookId Int @unique
}
```

의 형태을 띈다.
